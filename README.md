随着无畏契约的爆火闲鱼、拼多多、小红书等平台高价卖出的吸附外挂

其吸附外挂的工作流程可以拆解为以下四个自动化步骤：
A. 屏幕抓取 (Screen Capture)外挂程序以极高的频率（通常每秒 60-144 次，匹配显示器帧率）截取游戏画面。它不读取游戏内存，只是获取显示器输出的像素数据。
B. 目标检测 (Object Detection)这是 AI 外挂的“大脑”。它通常使用 YOLO (You Only Look Once) 系列轻量级神经网络模型。训练过程：开发者事先给成千上万张游戏截图打标签（标注出敌人的头部、身体）。实时推理：AI 在截图中搜索符合“敌人”特征的像素区域，并返回坐标 $(x, y)$。
C. 坐标计算 (Coordinate Transformation)AI 识别出敌人中心点后，程序会计算该点与屏幕中心（即你的准星）的相对距离。$$\Delta x = X_{target} - X_{center}$$$$\Delta y = Y_{target} - Y_{center}$$
D. 指令模拟 (Input Simulation)外挂向操作系统发送鼠标移动指令。为了逃避检测，高级外挂会加入**“平滑度” (Smoothing)**，让准星不是瞬间瞬移，而是像人手一样匀速划过去；或者使用硬件盒子（如 KMBox）伪装成真实的 USB 鼠标硬件信号。


通过作者自行研究，发现市面上主要是下面四种类型的吸附外挂，先将其开源，供大家研究。

V1：
找色识别吸附 仅支持紫色
会锁尸体会锁队友会锁技能
无配置要求 任何配置效果都是一样的
分辨率受限制 必须1920x1080 其它分辨率无效果

V2：
采用Ncnn推理的AI框架
不锁尸体不锁队友不锁技能 仅支持紫色
配置要求较高 显卡需达到20系以上效果才会好 CPU需达到11代以上效果才会好
常用分辨率全部支持 2k以及2k以下都能使用 3k以上不支持

V3：
采用Dml推理的AI框架
不锁尸体不锁队友不锁技能 支持黄色/紫色/红色
配置要求较低 显卡达到1060以上效果都好 CPU达到9代以上效果都好
常用分辨率全部支持 2k以及2k以下都能使用 3k以上不支持

V4：
采用Dml推理的AI框架
不锁尸体不锁队友不锁技能 支持自定义模型文件
只要你有onnx格式的模型文件，就能在任何游戏上运行
配置要求较低 显卡达到1060以上效果都好 CPU达到9代以上效果都好
常用分辨率全部支持 2k以及2k以下都能使用 3k以上不支持
支持 单机/双机使用 支持纯软件/硬件使用
这个版本预计也是最后一个版本了，以后有任何新功能都会在V4上进行添加
